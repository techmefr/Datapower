Initial release of DataPower

- Vue directives: `v-t-id`, `v-t-class`, `v-t-present`
- Composable: `useDatapower()` with `tAttrs`, `tId`, `tClass`, `tPresent`
- Build-time stripping of test attributes in production
- Environment-based configuration (development, test, staging, production)
- SSR compatible
